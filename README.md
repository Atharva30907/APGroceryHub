# 🛒 AP GroceryHub

**AP GroceryHub** is a responsive online grocery store built with **HTML5, CSS3, and JavaScript**. It features product browsing, search, cart management, user signup/login with `localStorage`, and a payment page with card and UPI/QRCODE options.

---

## ✨ Features

- **Home Page** – Product grid, search bar, and clean UI  
- **Cart Page** – Add/remove items, change quantity, view subtotal & total  
- **Signup & Login** – User account registration and login (stored in `localStorage`)  
- **Payment Page** – Card details form and UPI/QRCODE payment option  
- **Order Processing** – Simulated order creation and storage in `localStorage`

---

## 🚀 How to Run

1. Open the project folder in **VS Code**.
2. Use **Live Server** (or any local server) to open `index.html`.
3. Explore:
   - `signup.html` – Create an account
   - `login.html` – Login to your account
   - `cart.html` – View and manage your cart
   - `payment.html` – Complete the checkout flow

---

## 🛠️ Built With

- **HTML5** – Semantic structure and forms  
- **CSS3** – Responsive layout, flexbox, and modern styling  
- **JavaScript (vanilla)** – Cart logic, user auth, localStorage, and payment simulation
